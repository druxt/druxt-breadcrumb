DruxtJS Breadcrumb

Provides a Breadcrumb Vue component to be used within a Druxt (DRUpal nuXT) project.

Get started

The DruxtJS Breadcrumb module adds a Drupal (opens new window) Breadcrumbs Vue.js components to your Nuxt.js (opens new window) application.

# DruxtBreadcrumb

The DruxtBreadcrumb component uses the Vue Router and DruxtRouter to build a list of crumbs.
