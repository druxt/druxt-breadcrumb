# DruxtBreadcrumbMixin

Provides Vue.js properties to render Drupal Breadcrumb component.

Kind: global mixin
Example

<script>
import { DruxtBreadcrumbMixin } from 'druxt-breadcrumb'

export default {
  mixins: [DruxtBreadcrumbMixin],
}
</script>
